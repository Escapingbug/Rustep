# Rustep - Rust Executable Parser
A executable parser written in Rust. ELF support is now done, bugs are remain to be discovered and
fixed.

# Current Progress
* [x] `ELF` file format support
* [ ] `PE` file format support
* [ ] `macho` file format support
* [ ] DWARF support
* [ ] operation support
